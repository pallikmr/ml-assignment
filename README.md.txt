# Red Wine Quality Prediction

Machine Learning project using Streamlit.

## Folder Structure

project-folder/
│-- app.py  
│-- save_models.py  
│-- requirements.txt  
│-- README.md  
│-- red-wine.csv  
│-- model/

## Models Used

- Logistic Regression
- Random Forest
- XGBoost

## How To Run

1. Install packages

pip install -r requirements.txt

2. Save models

python save_models.py

3. Run app

streamlit run app.py

## Output

Predicts wine quality as Good or Poor.
